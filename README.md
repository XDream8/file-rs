    <p>a tool for determining file types, an alternative to file</p>
</div>

this is a **WIP**

## whats done

- [x] determining file extension
- [x] determining file type
- [x] determining file's mime type

## usage

```sh
$ file-rs -h
$ file-rs <files> <args>
```

- example:

```sh
$ file-rs Cargo.toml
```
